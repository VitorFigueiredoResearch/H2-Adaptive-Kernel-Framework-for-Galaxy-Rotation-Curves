═══════════════════════════════════════════════════════

  Interpolation: V_model interpolated onto the observed R grid
  Minimum points: n_inner >= 3 required for a valid scatter estimate
  Guard: velocities clipped to eps=1e-12 before log (prevents log(0))

3. DELTA-SIGMA DEFINITIONS (slight structural difference)
═══════════════════════════════════════════════════════

H2:   delta_sigma = sigma(V_H2_adaptive) - sigma(V_H1_baseline)
      → model-vs-baseline: one specific perturbation (H1→H2 adaptation)

NFW:  delta_sigma_* = sigma(V_NFW_perturbed) - sigma(V_NFW_baseline)
      → max_abs_delta_sigma = max over all perturbation magnitudes/signs
      → worst-case across ±10%, ±20% in M200, C200, and both

MOND: delta_sigma = sigma(V_MOND_perturbed) - sigma(V_MOND_baseline)
      → max_abs_delta_sigma = max over all perturbation magnitudes/signs
      → worst-case across ±10%, ±20% in g_dag, Ydisk, and both

ASSESSMENT:
  H2 reports a single specific delta (H1→H2 model change).
  NFW and MOND report worst-case sensitivity over multiple perturbation
  directions. These measure slightly different things:
    - H2 asks: "how much does this particular model change scatter?"
    - NFW/MOND ask: "what is the maximum scatter sensitivity to bounded
      parameter variations?"
  This is a structural comparison, not a unit or formula mismatch.
  The comparison is COMMENSURABLE but not IDENTICAL in experimental design.
  This distinction must be acknowledged in the manuscript.
